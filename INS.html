🗳️ Indian Election Advisor — Quick Guide

🚀 Getting Started

  • Tap ‘Let’s Get Started’
  • AI assistant greets you & lists available help

🎛 Interface Controls (Top-right)

  • • हिन्दी / English – Change language
  • • 🌙 / ☀️ – Dark/Light mode
  • • 🔇 / 🔊 – Voice on/off

📌 Quick Topics

  • 📝 Voter Registration
  • 🆔 EPIC Card
  • 📍 Polling Station
  • 📅 Election Schedule
  • 🗳️ EVM/VVPAT
  • ⚖️ Voter Rights

💬 Asking Questions

  • Text: Type in box → Press Enter / 📤 Send
  • Voice: 🎤 → Speak (Hindi/English)

✅ You Can Ask About

  • Registration & ID process
  • EPIC card apply/correct
  • Polling location info
  • Election dates & rules
  • EVM usage guide
  • Rights & responsibilities

❌ Not Covered

  • Non-election topics
  • Legal advice
  • Political endorsements

📞 Quick Help

  • Helpline: 1950
  • Website: eci.gov.in
  • Portal: nvsp.in

🔧 Troubleshooting

  • Voice not working → Allow mic permissions
  • Page errors → Refresh / clear cache
  • AI silent → Check internet
💡 Tip: Use keywords like ‘voter registration’ for best results.
